# Sustainable Discount Marketplace - BGT Market

## Project Overview
This project is a web-based **Sustainable Discount Marketplace** developed as part of the **CTIS 256 - Introduction to Backend Development** course at Bilkent University. The platform, branded as **BGT Market**, is designed to tackle food waste by providing a marketplace where users can purchase products near their expiration dates at significant discounts.

The application follows the Full-Stack development model, utilizing **Node.js** and **Express** for backend logic, **MySQL** for data management, and **EJS** for dynamic frontend rendering.

## Key Features
- **User Roles:** Distinct interfaces and functionalities for both Consumers and Market owners.
- **Sustainability:** A specialized system focusing on "near-expiry" product rescue to reduce environmental impact.
- **Secure Authentication:** Implementation of password hashing and automated email verification via **Nodemailer**.
- **Containerization:** Fully configured with **Docker** to ensure a consistent development environment across all services.
- **Dynamic Experience:** AJAX-supported shopping cart and responsive UI designed with pure HTML/CSS and JavaScript.

## Technical Stack
- **Backend:** Node.js, Express.js
- **Database:** MySQL (v8.0)
- **Templating Engine:** EJS
- **Infrastructure:** Docker & Docker Compose
- **Version Control:** Git & GitHub
